# О проекте
## Описание идеи
Идея заключается в том что используя современные технологии можно оживить рисунки ребенка, что поможет развитию его вооброжения. Каждый в детстве когда рисовал рисунок, представлял его, все что там происходит, и эта игра, реализовывает рисунки ребенка.
## Стек технологий, которые использовались в разработке
В качестве разработки использовался движок **Unity Engine**, языки программирования **C#**, **javaScript**.
## Описание компонентов системы
Система состоит из **Unity WebGL** (frontend)
